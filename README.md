## Getting Started

Install dependencies,

```bash
$ npm i 或 yarn
```

Start the dev server,

```bash
$ npm run start 或 yarn start
```

Build documentation,

```bash
$ npm run docs:build 或 yarn docs:build 每次文档更新时需执行此命令并提交至git仓库
```

Run test,

```bash
$ npm test
```

Build library via `father`,

```bash
$ npm run build 或 yarn build
```
